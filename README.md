# Futurize – Career Counseling Website.

## Overview

**Futurize** is a career counseling web application built using **HTML, CSS, JavaScript, and PHP**.
The purpose of the website is to help students get career suggestions based on their interests and answers to a quiz.

Users can create an account, log in, take a career quiz, and receive a suggested career path based on their responses.



## Features

* User **registration and login system**
* **Secure data storage** using a database
* **Career quiz system**
* **Career suggestions based on quiz answers**
* **Admin panel** for managing user data
* **Logout functionality**
* Simple and clean user interface


## How the Website Works

### 1. Registration

New users first create an account by registering on the website.
Their **username and password are stored in the database**.

### 2. Login

After registration, users can **log in using their credentials**.
Once authenticated, they are redirected to the main dashboard.

### 3. Career Quiz

Logged-in users can take a **career quiz** that contains multiple questions about their interests, preferences, and skills.

### 4. Answer Processing

The system evaluates the answers submitted by the user.
Based on predefined logic in the backend, the application determines which career path best matches the user's responses.

### 5. Career Suggestion

After submitting the quiz, the website **suggests a suitable career option** that aligns with the user's answers.

### 6. Admin Panel

The admin panel allows the administrator to:

* View registered users
* Access stored user data
* Manage basic website data

### 7. Logout

Users can log out of the system, which **ends the session and returns them to the login page**.
